# ReadQuest

초등 1~3학년 대상 독서 퀴즈 학습 도구 (React + Vite)

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속

## 로그인

- **테스트 비밀번호**: `readquest123`

## 화면 흐름

1. **로그인** → 비밀번호 입력 후 들어가기
2. **도서 목록** → 책 카드 형태, 무한 스크롤
3. **도서 선택** → 활동 타입 4가지(OX 퀴즈, 객관식, 순서 맞추기, 빈칸 채우기) 중 선택
4. **활동 플레이** → 첫 문제가 AI로 생성됨. 풀고 "다음 문제" 클릭 시 다음 문제가 한 개씩 생성. 더 만들 문제가 없으면 "모든 문제를 풀었어요!" 후 종료 (문제 개수 제한 없음)
5. **뒤로** 버튼으로 이전 화면 이동 (다시 진입 시 초기화)

## 콘텐츠 구성 (AI 생성)

- **OX 퀴즈**: 질문에 맞게 O / X 선택
- **객관식 퀴즈**: 네 보기 중 정답 선택
- **순서 맞추기**: 마우스 또는 터치로 드래그해 순서를 바꾼 뒤 확인
- **빈칸 채우기**: 문장의 빈칸에 들어갈 말을 보기에서 선택

문제와 답은 전부 AI가 도서 내용(`contentForPrompt`)을 바탕으로 생성합니다.

## AI / 환경 변수

- **VITE_OPENAI_API_KEY** (필수): OpenAI API 키. `.env` 파일에 설정하고, **절대 Git에 커밋하지 마세요.** (`.gitignore`에 `.env` 포함됨)
- `.env.example`을 참고해 `.env`를 만들고 키를 넣으면 됩니다.

## 기술 스택

- React 18, Vite 6, React Router 6, TypeScript
